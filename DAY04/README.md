# Day 4 - Model Evaluation Metrics

## Topics Covered

- Overfitting
- Underfitting
- Good Fit
- Imbalanced Datasets
- Confusion Matrix
- Accuracy
- Precision
- Recall
- F1 Score

---

## Key Learnings

- High accuracy does not always mean a good model.
- Overfitting causes poor performance on unseen data.
- Accuracy can be misleading on imbalanced datasets.
- Precision measures prediction correctness.
- Recall measures ability to find positives.
- F1 Score balances Precision and Recall.
