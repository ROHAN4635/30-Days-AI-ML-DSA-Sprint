# Day 1 - Setup, Git and First Dataset

## 1. What is Machine Learning?

Machine Learning is a field of AI where computers learn patterns from data and make predictions without being explicitly programmed.

---

## 2. What is a Dataset?

A dataset is a collection of data arranged in rows and columns.

Rows:
- Individual records/samples

Columns:
- Attributes/features

Example:

| Sepal Length | Sepal Width | Species |
|-------------|------------|----------|
| 5.1 | 3.5 | Setosa |

---

## 3. Features

Features are input variables used by the model for learning.

Example:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 4. Labels

Labels are the target outputs that the model predicts.

Example:
- Setosa
- Versicolor
- Virginica

---

## 5. Iris Dataset

Contains:
- 150 samples
- 4 features
- 3 classes

Classes:
- 0 = Setosa
- 1 = Versicolor
- 2 = Virginica

## 6. Git Basics

Initialize repository:

git init

Check status:

git status

Add files:

git add .

Commit changes:

git commit -m "message"

Push to GitHub:

git push
